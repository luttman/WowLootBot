Privacy Policy

Last updated: 2026-07-22

This page explains what WoWLootBot ("the bot") stores, why, and how to get it removed. It's written in plain language on purpose; if anything is unclear, use the contact details at the bottom.

What's stored

When a loot master runs /loot-add, the bot stores the contents of the export they upload:

  • World of Warcraft character names of who won each item
  • Item names and the in-game item ID
  • The raid date, boss name, and roll type (e.g. main spec, off spec) when the export includes them
  • Any note the loot council left on an award
  • The Discord username of whoever ran /loot-add, credited as the loot master for that raid unless overridden

The bot also stores a small amount of configuration per Discord server:

  • The server's Discord ID, so data from different servers never mixes
  • A role ID and/or channel ID, if a server sets one with /loot-config

What's not stored

The bot cannot read regular chat messages. It only sees the input of the slash commands listed on the home page. It never collects passwords, payment information, emails, or Discord account details beyond a username and the IDs listed above.

Where it's stored

Data lives in a single database operated for this bot. It is not sold, and it is not shared with advertisers or other third parties. Fill in details here if you use a specific hosting provider you want to disclose.

How long it's kept

Loot history is kept until a server's loot master deletes it with /loot-clear, or until the bot is removed from the server, at which point that server's data is deleted within a reasonable period. There's no automatic expiry before then, since the point of the bot is to keep a long-term record.

Your choices

  • Any loot master can delete a single raid or an entire server's history at any time with /loot-clear.
  • A server admin can restrict who is allowed to add or delete loot data with /loot-config.
  • Removing the bot from your server stops all data collection immediately.

Changes to this policy

If what's collected or how it's used changes meaningfully, this page will be updated and the date at the top will change.

Contact

To ask about stored data or request deletion outside of /loot-clear: contact@wowlootbot.xyz
